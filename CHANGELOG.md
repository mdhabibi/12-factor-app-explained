# Changelog

All notable changes to this tutorial are documented here.
This project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] — 2026-06-03

First public release. 🎉

### Added
- Complete beginner-friendly tutorial covering all 12 factors, each with an analogy, a Python "don't vs. do" example, callouts, and a "why it matters."
- 11 inline SVG diagrams illustrating the trickier concepts (config-per-environment, stateless processes, build/release/run, scaling, disposability lifecycle, logs as a stream, and more).
- "Modern context" section mapping each factor to Docker, Kubernetes, and CI/CD, plus the "15-factor" extensions (API first, Telemetry, Security) and a health-checks bonus.
- One-page cheat sheet and a 40+ term glossary.
- Single self-contained `index.html` — no build step or dependencies.
- Published as a live site via GitHub Pages.
